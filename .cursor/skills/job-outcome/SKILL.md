# Job Outcome

## Quick Start

Follow `WORKFLOW.md`. Update `job_search_tracker.csv` and archive materials under `documents/applications/<company>_<role>/`. This skill records what happened; it does not recalibrate the profile directly.

Never reconstruct a dead posting from memory. Fetch it, use a user-pasted copy, or mark it unavailable.
