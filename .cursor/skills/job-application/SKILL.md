# Job Application

## Quick Start

For a job URL or pasted posting, follow `WORKFLOW.md` in order: parse, evaluate fit, ask to proceed, draft, review, revise, compile, inspect PDFs, run ATS checks, and present final files.

For partial requests, use the relevant reference:
- Evaluate-only: `references/04-job-evaluation.md` and `references/01-candidate-profile.md`.
- CV drafting: `references/05-cv-templates.md`.
- Cover letter drafting: `references/03-writing-style.md` and `references/06-cover-letter-templates.md`.
- Interview prep: `references/07-interview-prep.md`.

## Non-Negotiables

- Never draft before a fit evaluation and user approval.
- Never invent profile or company facts.
- Compile and inspect PDFs before final delivery.
- Use `Cursor` or `AI-assisted development` for AI tooling references unless the profile says otherwise.
